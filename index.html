Biography

I am a PhD candidate at the Institute of Applied Information Processing and Communications at Graz University of Technology. I focus on efficient software isolation for sandboxing, enclaves, and memory safety.

Interests

  • System Security
  • Memory Safety
  • Sandboxing
  • Confidential Computing / Shielded Execution

  • Low-Level Software
  • Processor Design
  • Applied Cryptography
